# 🏥 MediTrack

MediTrack is a secure and user-friendly web application for managing personal medical records. Built with Node.js, Express.js, MongoDB, and EJS, it allows users to upload, organize, and share medical files with ease.

---

## 🚀 Features

- 🔐 **User Authentication** – Secure login & registration using hashed passwords
- 📁 **Upload & Manage Records** – Upload medical documents (PDFs, images)
- 🗂️ **Organized Dashboard** – View, download, or delete uploaded files
- 📤 **Email Sharing** – Send medical records directly to doctors via email
- 💡 **Intuitive Interface** – Clean and responsive UI with EJS templating

---

## 🛠️ Tech Stack

| Layer         | Technology            |
|---------------|------------------------|
| Backend       | Node.js, Express.js    |
| Frontend      | EJS (Embedded JS)      |
| Database      | MongoDB (via Mongoose) |
| File Uploads  | Multer                 |
| Email Service | Nodemailer             |
| Styling       | HTML, CSS              |
